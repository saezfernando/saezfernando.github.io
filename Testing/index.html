Inicio

"The software testing is the process of establishing confidence in that the program or system does what it should do." [Hetzel 73]

"The software testing is the process of executing a program or system with the intention of finding errors.“ [Myers 79]

“The process of operating a system or component under specified conditions, observing or recording the results, and making an evaluation of some aspect of the system or component”. [IEEE 90]
